Here is a list of HTML recourses if you want to become a web developer


HTML for beginners- this is a free course by CodeCademy. You can receive a certificate when you do the paid course.
HTML fundamentals and Updates- This course is on Alison, a website where courses are free. This course will show you the fundamentals of HTML and help understand HTML.
Udemy - Mark Lassoff Web Development Masterclass- This course is my ultimate favorite. They cover everything that's important on how to become a web developer, including HTML5 fundamentals.
HTML crash course for beginners- Learn HTML for a career in web development. This HTML tutorial teaches you everything you need to get started. It's created by Programming with Mosh on YouTube
Html basics for a basic website- FreeCodeCamp teaches you how to build a basic website in just 15 minutes using basic HTML.
w3schools free tutorials on HTML- w3schools have a variety on html fundamentals. They also have a 'try it yourself' section where you can practice.
Downloadable PDF from Tutorialspoint- This PDF contains everything you would want to know on HTML. It covers everything on HTML.
HTML website- HTML itself? Who knows, but they have some pretty awesome stuff and information there as well. PS they have a cheat-sheet as well... shhh!
List of all the HTML codes and explanations- digital.com has a list of all the codes and they explain it aswell, adding a picture for you to understand what they're explaining.
Book: Beginners Guide to HTML- This book is written by Wendy Willard. This book covers everything you want to know about HTML. If you're a bookworm, this will definitely be your cup of tea.

Resources are easily accessible. It's up to you if you want to take the chance to learn it.
